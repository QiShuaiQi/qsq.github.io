通过本章学习，您将可以了解基本的SELECT语句

  • 基本的SELECT语句
  • 过滤和排序数据
  • 分组函数
  • 分组查询
  • 多表查询
  • 分页查询

    SELECT 语句

  • select *from
  • select 标识选择那些列。
  • *表示选择所有字段
  • from 表示从那些表中选择。
  • 选择特定的列把字段标注出来写在select 之后

    注 意

  • SQL 语言大小写不敏感。
  • SQL 可以写在一行或者多行
  • 关键字不能被缩写也不能分行
  • 各子句一般要分行写。
  • 使用缩进提高语句的可读性。

    列的别名

  • 重命名一个列。
  • 便于计算。
  • 紧跟列名,也可以在列名和别名之间加入关键字**‘AS’，别名使用双引号，以便在别名中包含 空格或特殊的字符并区分大小写。
  • select 字段 AS “别名”

字符串

  • 字符串可以是 SELECT 列表中的一个字符,数字,日
  • 日期和字符只能在单引号中出现。
  • 每当返回一行时，字符串被输出一次。

显示表结构

使用 DESCRIBE 命令，表示表结构 desc 表名

过滤

使用WHERE 子句，将不满足条件的行过滤掉。
WHERE 子句紧随 FROM 子句
select * from table where = 字段

比较运算

操作符 含义
= 等于 (不是 ==)
> 大于
>= 大于、等于
< 小于
<= 小于、等于
<> 不等于 (也可以是 !=)

其他比较运行

操作符 含义
BETWEEN …AND… 在两个值之间 (包含边界)
IN(set) 等于值列表中的一个
LIKE 模糊查询
IS NULL 空值

BETWEEN

  • 使用 BETWEEN 运算来显示在一个区间内的值

  • select 字段 from 表名 where 字段 值 AND值

IN

  • 使用 IN运算显示列表中的值。
  • select 字段 from 表名 where 字段 IN(值，值，值) LIKE

LIKE

  • 使用 LIKE 运算选择类似的值
  • 选择条件可以包含字符或数字：
  • % 代表零个或多个字符(任意个字符)。
  • _ 代表一个字符。
  • select 字段 from 表名 where 字段 like ‘s’;
  • 同时 ‘%’和‘-’可以同时使用。
  • select 字段 from 表名 where 字段 like ‘_o%’

NULL

  • 使用 IS (NOT) NULL 判断空值。
  • select 字段 from 表名 where 字段 IS NULL

逻辑运算

操作符 含义
AND 逻辑并
OR 逻辑或
NOT 逻辑否

AND

  • AND 要求并的关系为真。
  • select 字段 from 表名 where 条件一 AND 条件二 要求都符合

OR

  • OR 要求或关系为真
  • select 字段 from 表名 where 条件一 OR 条件二 只要其中一个符合

NOT

  • select 字段 from 表名 where 字段名 NOT IN (‘值’,’值’,’值‘) 返回不包含括号里的值

ORDER BY子句

  • ASC（ascend）: 升序 select 字段 from 表名 order by 字段 ASC
  • DESC（descend）: 降序 select 字段 from 表名 order by 字段 DESC
